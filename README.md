# Highway Run - Greenfoot Game


![Demo](gif/HighwayRun.gif)


Ένα 2D top-down racing game, βασισμένο στην γλώσσα Java και στις αρχές του Αντικειμενοστρεφής Προγραμματισμού. Αναπτύχθηκε στο περιβάλλον του Greenfoot.


Ο παίκτης οδηγεί ένα αυτοκίνητο σε έναν δρόμο ταχείας κυκλοφορίας, αποφεύγοντας εμπόδια και προσπαθώντας να επιτύχει το υψηλότερο δυνατό σκορ. Το παιχνίδι διαθέτει 2 διαφορετικά επίπεδα, το καθένα με την δική του ξεχωριστή κλιμακούμενη δυσκολία.
